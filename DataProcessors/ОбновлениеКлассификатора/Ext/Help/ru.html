Обработка "Обновление классификатора" предназначена для обновления "Классификатора банков РФ", "Классификатора валют" и "Классификатора единиц измерения" и элементов справочников "Банки", "Валюты" и "ОКЕИ".

Обновление классификатора банков, валют и единиц измерения производится с диска "Информационно- технологическое сопровождение "1С:Предприятие" (начиная с октября 2000 года), кроме того классификатор банков можно обновлять с web-сайта агентства "РосБизнесКонсалтинг" (www.rbc.ru). Для обновления классификатора установите флажок "Загрузить классификатор ".

Установите переключатель откуда будет производиться загрузка (обновление).

Для того, чтобы привести элементы справочника в соответствие с классификатором, установите флажок "Обновить элементы справочника".

Для запуска процесса обновления нажмите кнопку "Выполнить".

Если установлен флаг "Обновить элементы справочника", обработка проверит существующие элементы справочника. Если будут найдены элементы, чьи реквизиты не соответствуют "Классификатору", или отсутствующие в нем элементы, будет показан список таких элементов, с описанием отличий. В списке можно снять пометки у тех элементов, которые не нужно обновлять.

Для обновления элементов справочника следует вновь нажать кнопку "Выполнить".
