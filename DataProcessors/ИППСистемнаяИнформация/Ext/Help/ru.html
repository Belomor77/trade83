Большинство реквизитов заполняется программой автоматически на основании опроса данных компьютера, программы и базы данных. Вам необходимо лишь внести недостающие или исправить любой из имеющихся.

Перечень используемых реквизитов и их краткое описание: Сведения об используемом программном обеспечении Программа * - Полное наименование программы по регистрационной анкете.

Релиз программы * - Чтобы узнать номер релиза программы, выберите пункт "О программе" раздела меню "Помощь". Номер указывается в правом верхнем углу окна.

Конфигурация * - Чтобы узнать наименование конфигурации, выберите пункт "О программе" раздела меню "Помощь". Наименование конфигурации указывается ниже слова "Конфигурация". Также наименование конфигурации приводится на регистрационной анкете.

Релиз конфигурации * - Чтобы узнать номер релиза конфигурации, выберите пункт "О программе" раздела меню "Помощь". Номер указывается в правом верхнем углу окна, ниже номера релиза программы.

Рег. номер * - Регистрационный номер указан на коробке (штрих-код), на документации и на регистрационной анкете.

Изменения - Примерная степень изменения типовой конфигурации.

Сведения об организации, в которой используется данная программа Организация * - Наименование организации, использующей программу Ответственный * - Фамилия ответственного за работу с программой лица Телефон - Контактный телефон Факс - Номер факса Для писем с вопросами и описанием ошибок желательно указать сведения о компьютере Компьютер - Тип компьютера (процессора) Быстродействие - Быстродействие процессора Память - Объем оперативной памяти в Мб Опер. система - Операционная система Сеть - Используемая сеть Реквизиты, помеченные символом * являются обязательными для заполнения (передача писем возможна только при заполненных обязательных реквизитах).
