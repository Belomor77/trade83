Окно ввода адреса используется для ввода адреса организации, адреса постоянного места жительства физического лица и адреса строящегося объекта, по которому физическому лицу произведены вычеты из налогооблагаемой суммы.

Перед заполнением адресов необходимо загрузить таблицы с классификаторами в специальную папку.

Для классификатора формата 2002 года требуются следующие таблицы: Kladr.dbf Адресный классификатор используется для заполнения адресных полей региона, района, города и населенного пункта.

Street.dbf Классификатор улиц используется для правильного заполнения адресных полей улицы и индекса.

Классификатор домов используется для заполнения индекса.

Socrbase.dbf Классификатор сокращений используется для правильной расшифровки адресных сокращений.

Для классификатора формата 2003 года дополнительно требуется таблица Doma.dbf Классификатор домов используется для поиска индекса по номеру дома.

В формате 2003 года предусмотрено дополнительно хранение уже устаревших на текущий момент адресных сведений, например, город Ленинград. В обработке предусмотрен выбор: показывать все адресные сведения или только значимые адресные сведения. Для настройки такого фильтра нужно установить, либо снять флажок "Показывать только значимые адреса (для формата 2003 г)".

При изменении формата классификатора или фильтра при следующем выборе адреса программа предложит произвести переиндексирование классификаторов.
