Обращение можно отправить в том случае, если у Вас есть доступ в Интернет.

Доступ осуществляется стандартными средствами MAPI.

Описание действий при заполнении обращения 1. Указание сведений о программе и пользователе.

Для правильной обработки обращения необходимо указать различные сведения (об используемой программе, пользователе и компьютере). Для ввода сведений дважды щелкните мышью ячейку "Сведения", расположенную в верхней строке формы.

Многие параметры программа определит самостоятельно, но их можно исправить или стереть, если не хотите их сообщать. Обязательными к заполнению являются параметры, отмеченные символом "*".

2. Ввод многострочного текста.

Для создания нового абзаца при написании обращения используйте сочетание клавиш CTRL+ENTER.

Нажатие клавиши ENTER завершает ввод текста мнения.

Чтобы продолжить ввод текста нажмите клавишу ENTER. При этом весь текст выделяется.

Для снятия выделения нажмите любую клавишу управления курсором или щелкнуть мышью в выделенном тексте. Далее можете продолжить ввод текста.

3. Сохранение введенной таблицы.

Если требуется сохранить всю таблицу, выберите пункт "Сохранить как" раздела меню "Файл".

Укажите имя файла и тип ("Таблицы (*.mxl)").

4. Печать таблицы.

Для печати таблицы выберите пункт "Печать" раздела меню "Файл".

Укажите принтер и количество экземпляров при печати.

Для настройки отступов выберите пункт "Параметры страницы".

Если не используется цветной принтер, рекомендуем указать режим "Черно-белая печать".

5. Отправка обращения Для отправки обращения дважды щелкните мышью ячейку "Отправить".
