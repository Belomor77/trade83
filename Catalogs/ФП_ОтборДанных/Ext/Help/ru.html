Справочник "Настройка отбора данных для планирования" Данный справочник предназначен для задания правил отбора данных, выгружаемых в конфигурацию "Финансовое планирование". Справочник подчинен справочнику "Настройка выгрузки данных для планирования"; отбор настраивается отдельно для каждого из элементов справочника-владельца.

Справочник открывается по нажатию кнопки "Отбор итогов" в форме элемента или списка справочника "Настройка выгрузки данных для планирования".

Справочник имеет простую одноуровневую структуру. Настойка выгрузки производится путем вода условий отбора данных. Каждое условие вводится как отдельный элемент справочника.

Редактирование элементов справочника производится в форме списка или в отдельном диалоговом окне.

Параметр. Здесь необходимо указать один из параметров отбора. В зависимости от настройки владельца - элемента справочника "Настройка выгрузки данных для планирования" это могут быть атрибуты бухгалтерских итогов или регистров оперативного учета. Поле обязательно должно быть заполнено.

Операнд. Здесь необходимо выбрать операнд условия: "Равно" или "Не равно". Поле обязательно должно быть заполнено.

Значение. Здесь необходимо выбрать значение, которому должен быть равен или не равен указанный параметр. В зависимости от параметра, это должен быть один из объектов, обычно элемент справочника или документ конфигурации, в которую встроен механизм выгрузки.
