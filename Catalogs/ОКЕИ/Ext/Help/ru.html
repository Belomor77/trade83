В этот справочник вводится список используемых компанией типов единиц измерения (штуки, контейнеры, ящики, коробки и т.д.), из которого будет выбираться тип базовой единицы измерения для каждого товара или услуги и тип единицы измерения для каждой упаковки товара. Информация в справочник заносится в виде списка.
