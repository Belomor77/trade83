Справочник "Договоры" является подчиненным справочнику "Контрагенты".

После записи информации в форме контрагента на закладке "Договор взаиморасчетов" данные об основном договоре контрагента будут автоматически перенесены в этот справочник.

В окне списка договоров можно посмотреть основные реквизиты договора, состояние взаиморасчетов по договору ("Долг").

С помощью кнопки "Итоги" можно вывести отчет по взаиморасчетам с контрагентом по договору с детализацией по документам.

Редактирование реквизитов договора производится в отдельном диалоговом окне.

Для каждого контрагента на закладке "Договор взаиморасчетов" определяется основной договор.

Параметры основного договора могут быть заполнены непосредственно в форме контрагента.

Указание договора в документе является обязательным. По умолчанию в документе устанавливается основной договор, определенный для контрагента. В любой момент времени в документе может быть выбран, любой другой договор.

В договоре определяются валюта взаиморасчетов с контрагентом по договору.

В этой валюте будут фиксироваться все взаиморасчеты с контрагентом при проведении документов в рамках данного договора.

В договоре определяются кредитные отношения с контрагентом.

Для договора определяется сумма и срок предоставляемого кредита.

Сумма кредита задается в валюте взаиморасчетов по договору.

В том случае, если по договору не ведется операций, связанных с отгрузкой или покупкой товара в кредит, то можно отключить контроль кредита (установить флаг в поле "Не контролировать кредит").

Если для договора установлен контроль кредита, проведение документов зависит от полномочий, установленных пользователю (справочник "Пользователи").

Если пользователю назначены полномочия (справочник "Полномочия пользователей"), разрешающие отпускать товар при превышении кредита, то такой документ будет проведен, при этом будет выдано предупреждающее сообщение.

Для каждого договора, оформленного с контрагентом, могут быть назначены индивидуальные цены и скидки (поля "Тип цен по умолчанию" и "Скидка по умолчанию"), которые будут автоматически заполняться при выборе данного договора в документе.

Для более детального анализа итогов взаиморасчетов в разрезе договоров в отчетах конфигурации добавлены возможности фильтрации и группировки данных по договорам.
